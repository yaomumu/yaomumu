## 🔬 一、传感器与信号处理算法

> **核心方向**：工业场景下的"数据采集 → 信号解析 → 风险监测/健康管理"全链路解决方案

### 1.1 铁路货车动态称重监测系统

#### 📋 项目背景
铁路货车超载是导致钢轨损耗加速、列车运行安全隐患增加的重要因素。传统静态称重存在效率低、无法覆盖全路网的问题，需要在列车运行过程中实现高精度动态称重。

#### 🎯 技术挑战
- **环境干扰**：温度波动、电磁干扰导致传感器数据漂移
- **动态测量**：列车高速通过时的振动影响测量精度
- **精度要求**：需达到误差≤2%的工业标准

#### 💡 解决方案
```
技术架构：
光纤光栅传感器(FBG) → 信号预处理(滤波/归一化) → 
粒子群算法(PSO)优化 → BP神经网络回归 → 动态称重结果
```

**核心技术栈**：
- 传感技术：光纤光栅传感器(Fiber Bragg Grating, FBG)
- 优化算法：粒子群算法(PSO)参数自适应调优
- 预测模型：BP神经网络非线性回归
- 信号处理：自适应滤波、温度补偿、数据归一化

#### ✅ 关键成果
- **精度指标**：80%样本的检测误差 ≤ 2%
- **抗干扰能力**：温度补偿算法使传感器温漂影响降低65%
- **实时性能**：单节车厢称重响应时间 < 0.5秒

### 1.2 机床刀具PHM系统 

#### 📋 项目背景
传统机床维护依赖"定期更换"或"经验判断"，存在两大痛点：
- **突发故障停机**：无预警的刀具崩刃导致生产中断，损失大
- **过度维护浪费**：未到寿命即更换，刀具成本浪费15-20%

本项目构建基于多传感器融合的PHM系统，实现**"监测→诊断→预测→决策"**全生命周期管理。

#### 🎯 技术挑战
- **复杂工况**：高速切削、材料硬度变化导致信号特征复杂
- **多故障模式**：磨损、崩刃、裂纹等5类故障特征相互耦合
- **实时预测**：需在生产过程中完成RUL预测，响应时间 < 5秒

#### 💡 解决方案

**系统架构**：
```
[多传感器数据采集层]
    ↓
声发射传感器 → 高频信号(100kHz-1MHz) → 微裂纹/摩擦特征
振动传感器   → 中频信号(0-10kHz)      → 宏观振动/失稳特征  
温度传感器   → 低频信号(0-10Hz)       → 热变形特征
    ↓
[信号预处理层]
小波去噪 → 带通滤波 → 归一化 → 特征提取(时域+频域+时频域)
    ↓
[智能诊断层]
SVM分类器 + 随机森林集成 → 5类故障识别 (准确率94%)
    ↓
[寿命预测层]
LSTM时序模型 + ARIMA统计模型 → 融合预测RUL (误差≤7%)
    ↓
[决策支持层]
维护建议生成 + 报警推送 (提前36小时预警)
```

**核心技术栈**：
- **多传感器融合**：声发射(AE) + 振动 + 温度三维信息互补
- **特征工程**：
  - 时域：RMS(均方根)、峰值因子、波形因子
  - 频域：谱峭度、频带能量比、主频漂移
  - 时频域：STFT短时傅里叶变换、小波包分解
- **故障诊断**：SVM(支持向量机) + 随机森林多分类器融合
- **寿命预测**：LSTM(长短期记忆网络) + ARIMA时序混合模型

#### ✅ 关键成果

**技术指标**：
| 指标类别 | 具体指标 | 性能表现 |
|---------|---------|---------|
| 故障识别 | 5类故障(磨损/崩刃/裂纹/粘结/烧伤)准确率 | ≥ 94% |
| 寿命预测 | RUL预测误差 | ≤ 7% |
| 预警时效 | 提前预警时间 | 36小时 |
| 传感器优势 | 相比单传感器特征覆盖率提升 | +35% |
| 复杂工况 | 高速切削场景识别稳定性提升 | +20% |

**技术亮点**：
1. **多传感器互补机制**：AE传感器捕捉早期裂纹，振动传感器监测宏观失稳，温度传感器评估热损伤
2. **混合预测模型**：LSTM擅长长期趋势，ARIMA擅长短期波动，融合后预测稳定性提升22%
3. **自适应阈值**：根据刀具型号、材料、工艺参数自动调整诊断阈值

### 1.3 油气管道振动监测与风险定位系统

#### 📋 项目背景
长距离油气管道(数百公里)周边施工、盗采等外部干扰是管道泄漏事故的主要诱因。传统人工巡检无法覆盖全线，需要实时监测+精准定位技术。

#### 🎯 技术挑战
- **长距离监测**：单根光纤覆盖50km管道，信号衰减严重
- **事件识别**：挖掘、施工、车辆通行等振动信号特征相似
- **精准定位**：需达到5米定位精度，误报率 < 3%

#### 💡 解决方案

**技术架构**：
```
分布式光纤传感器(DAS) → 振动信号采集(20kHz采样) →
STFT+CWT时频特征提取 → 深度学习分类(CNN+LSTM) →
双树小波定位算法 → 风险事件+位置输出
```

**核心技术栈**：
- 传感技术：分布式光纤声学传感(DAS, Distributed Acoustic Sensing)
- 时频分析：STFT(短时傅里叶变换) + CWT(连续小波变换)融合
- 纹理特征：GLCM(灰度共生矩阵)提取时频图二维特征
- 深度模型：CNN提取空间特征 + LSTM建模时序依赖

#### ✅ 关键成果
- **识别准确率**：8类典型事件(挖掘/施工/盗采等)识别率 > 92%
- **定位精度**：误差 ≤ 5米(50km监测范围内)
- **响应时间**：从事件发生到报警推送 < 3秒

### 2.1 铁路列车前方障碍物与司机行为识别系统

#### 📋 项目背景
铁路列车运行过程中存在瞭望盲区(弯道、夜间)，异物侵限(行人/车辆/土档)无法及时发现，紧急制动距离不足导致事故风险。

#### 🎯 技术挑战
- **远距离检测**：需在1000m+距离识别人员，为司机预留足够反应时间
- **复杂场景**：弯道遮挡、恶劣天气(雨雪雾)、夜间低照度
- **实时性要求**：处理延迟 < 100ms，帧率 ≥ 20fps

#### 💡 解决方案

**技术演进路径**：
```
阶段一(传统方法):
Canny/Sobel边缘检测 + Otsu阈值分割 + LBP/GLCM纹理特征
    ↓ (准确率70%，误检率高)
阶段二(深度学习):
YOLOv5目标检测 + 多帧时序验证 + RK3588嵌入式加速
    ↓ (准确率90%+，实时性提升3倍)
阶段三(融合定位):
图像检测 + GPS轨迹数据 → 障碍物地理位置标记
```

**核心技术栈**：
- **传统方法**：Canny边缘检测、Otsu自适应阈值、LBP局部二值模式、GLCM灰度共生矩阵
- **深度学习**：YOLOv5s轻量化模型、数据增强(光照/遮挡/尺度变换)
- **嵌入式部署**：RK3588 NPU加速、模型量化(FP16→INT8)
- **多模态融合**：图像检测结果 + GPS定位 + 轨道数据库

#### ✅ 关键成果
| 目标类型 | 最大检出距离 | 检出率 | 误检率 |
|---------|------------|-------|--------|
| 人员 | 1000m | ≥ 90% | < 5% |
| 车辆 | 1400m | ≥ 92% | < 4% |
| 大型障碍物 | 1600m | ≥ 95% | < 3% |

**特殊场景适配**：
- 弯道场景：多帧跟踪算法，连续3帧检出后触发报警
- 恶劣天气：雨雪天气误检率控制在8%以内(结合雷达融合系统)
- 夜间低照度：红外补光 + 图像增强算法

**现场验证**：已在**真实列车环境**完成10000公里测试，零误报导致的紧急制动

### 2.2 光伏电池板缺陷视觉检测系统

#### 📋 项目背景
光伏电池板在生产、运输、使用过程中易出现裂纹、污染、阴影遮挡等缺陷，导致发电效率下降5-15%。传统人工巡检效率低、漏检率高。

#### 🎯 技术挑战
- **缺陷类型多样**：裂纹、断栅、热斑等3类缺陷
- **细小特征**：部分裂纹宽度 < 0.5mm，需亚像素级定位
- **现场环境**：光照变化、反光干扰、安装角度差异

#### 💡 解决方案

**技术架构**：
```
[图像采集] 
工业相机(5MP) + 偏振镜(消除反光) + 均匀光源
    ↓
[预处理]
灰度化 → 直方图均衡化 → 去噪(高斯滤波)
    ↓
[缺陷检测分支]
分支一: U-Net语义分割 → 裂纹/热斑像素级定位
分支二: YOLOv5目标检测 → 污染/碎片/断栅边界框识别
    ↓
[后处理]
形态学操作(闭运算) → 连通域分析 → 缺陷分类+定位
    ↓
[结果输出]
缺陷类型 + 位置坐标 + 严重程度评级
```

**核心技术栈**：
- **语义分割**：U-Net(适用裂纹连续区域)、Mask R-CNN(兼顾检测+分割)
- **目标检测**：YOLOv5s + 锚框优化(适配细长裂纹)
- **数据增强**：旋转(0-360°)、随机裁剪、光照调整、噪声注入
- **嵌入式部署**：RK3588平台、模型剪枝(参数量减少40%)

#### ✅ 关键成果
- **定位精度**：缺陷空间定位误差 ≤ 2cm(电池板尺寸1.6m×1.0m)
- **分类准确率**：6类缺陷综合准确率 > 88%
- **检测速度**：单块电池板检测时间 < 2秒(含图像采集)

## 🧠 三、智能算法应用

> **核心方向**：多源数据融合，提升复杂场景下的系统鲁棒性与决策智能化

### 3.1 列车前方障碍物"图像+雷达"融合检测系统

#### 📋 项目背景
单一视觉系统在恶劣天气(大雾/雨雪)下误检率高达20%，需结合毫米波雷达3D点云数据进行交叉验证，确保安全性。

#### 🎯 技术挑战
- **多模态对齐**：图像像素坐标与雷达3D点云坐标系的精确标定
- **时序同步**：相机30fps、雷达10Hz，需时间戳对齐
- **决策融合**：两个模态判断冲突时的仲裁机制

#### 💡 解决方案

**融合架构**：
```
[感知层]
图像分支: 相机 → YOLOv5检测 → 二维边界框 + 置信度
雷达分支: 毫米波雷达 → 点云聚类 → 三维位置 + 速度
    ↓
[标定层]
联合标定(棋盘格+角反射器) → 图像-雷达坐标映射矩阵
    ↓
[匹配层]
3D点云投影到2D图像 → IoU匹配 → 对应目标关联
    ↓
[融合层]
规则一: 双模态均检出 → 高置信度输出
规则二: 仅图像检出+雷达无回波 → 降低置信度(可能误检)
规则三: 仅雷达检出+图像未检出 → 触发二次验证
    ↓
[决策层]
综合置信度阈值判断 → 障碍物真实性评估
```

**核心技术栈**：
- **图像检测**：YOLOv5 + 多帧跟踪(DeepSORT算法)
- **雷达处理**：DBSCAN点云聚类、卡尔曼滤波轨迹预测
- **多模态融合**：加权融合策略、贝叶斯决策理论
- **标定技术**：PnP算法、张正友标定法扩展

#### ✅ 关键成果
- **恶劣天气适应性**：大雾场景误检率降低40%，雨雪天气降低35%
- **真实性判断**：障碍物真实性判断准确率 > 95%(相比单模态提升25%)
- **距离精度**：融合后距离测量误差 ≤ 0.5米(0-100米范围)

#### 📋 项目背景
维修手册数量庞大，维修人员现场排故需大量时间翻阅文档，效率低且易遗漏关键信息。

#### 🎯 技术挑战
- **知识抽取**：从非结构化维修手册中提取"故障-原因-解决方案"三元组
- **语义理解**：维修人员问题描述口语化,需理解专业术语+自然语言混合表达
- **知识推理**：复杂故障需多跳推理(故障A→可能原因B→检查部件C→解决方案D)

#### 💡 解决方案

**系统架构**：
```
[知识构建]
命名实体识别(NER) → 关系抽取(RE) → Neo4j知识图谱存储
实体类型: 故障现象、故障原因、检查步骤、解决方案、飞机部件
关系类型: 导致、检查、解决、属于、关联
    ↓
[问答理解]
用户问题 → BERT语义编码 → 意图识别 + 实体识别 →
查询语句生成(Cypher)
    ↓
[知识检索]
图数据库查询 → 多跳路径搜索 → 候选答案召回
    ↓
[答案排序]
BERT相似度打分 → 历史采纳率加权 → Top-3答案返回
    ↓
[交互优化]
用户反馈 → 强化学习优化排序策略
```
